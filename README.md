## What Is This

THUNDERWIRE is a professional X (Twitter) content system embedded in a single file. It runs inside any AI — Claude Code, Cursor, Windsurf, ChatGPT, Gemini, or any API-based tool — and transforms any input (a topic, a niche, even just "find me something viral") into algorithmically engineered, human-tone content with built-in engagement strategy.

It is not a prompt. It is not a template generator. It is a structured 8-phase decision-making protocol built on:
- X's open-sourced algorithm code (github.com/xai-org/x-algorithm, January 2026)
- The confirmed Grok/Phoenix transformer engagement weights
- Peer-reviewed psychological research on viral content triggers
- Real audience language extraction methodology
- Professional anti-AI reconstruction — not just a word list

| Phase | Name | Function |

## How to Use

### Step 1: Download THUNDERWIRE.md

### Step 2: Activate in your AI

| Tool | How |
|------|-----|
| Claude Code | Copy to project folder, rename to CLAUDE.md, run `claude` |
| Cursor | Settings → Rules → paste full file contents |
| Windsurf | Create `.windsurfrules`, paste full file contents |
| ChatGPT | Settings → Custom Instructions → paste full file contents |
| Gemini | First message of session: paste full file contents |
| Any API | Pass full file as `system` parameter |

### Step 3: Set up your account

Tell THUNDERWIRE your niche in plain language:
> "My niche is [X]. My audience is [Y]. Run setup."

THUNDERWIRE asks 9 questions, then configures itself.

### Step 4: Create content

Just talk naturally:
- "Find a viral topic in [niche] and write an article"
- "Write a thread about [topic]"
- "I want to post something about [X] today"
- "What angle should I take on [Y]?"

THUNDERWIRE detects your intent and runs the correct pipeline phases automatically.

## The Anti-AI System

Every piece of content runs through a mandatory 5-layer reconstruction before delivery:
1. Entropy scan — detect uniform sentence structure (AI signature)
2. 47 Forbidden Terms scan — remove all AI vocabulary patterns
3. Opinion density audit — real takes, no hedging
4. Specificity reconstruction — concrete details replace abstractions
5. Cognitive marker injection — human thinking patterns inserted

Minimum human score: 7/10. Below that, it rebuilds, not delivers.
