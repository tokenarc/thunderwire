## What Is This

THUNDERWIRE is a professional X content intelligence protocol embedded in a single file. Drop it into any AI — Claude Code, OpenCode, Cursor, Windsurf, ChatGPT, Gemini, or any API — and it transforms any input into algorithmically engineered, human-tone content with built-in engagement strategy.

Not a prompt. Not a template generator. A structured **12-phase decision-making protocol** built on:

- X's open-sourced algorithm code (github.com/xai-org/x-algorithm, January 2026)
- The confirmed Grok/Phoenix transformer engagement weights
- Peer-reviewed psychological research on viral content triggers
- Real audience language extraction methodology
- Professional 7-layer anti-AI reconstruction — defeats GPTZero, Originality.ai, Turnitin, Copyleaks, Winston AI, ZeroGPT, and Content at Scale

## Quick Start

### Step 1: Get THUNDERWIRE.md

```bash
curl -sL https://raw.githubusercontent.com/tokenarc/thunderwire/main/THUNDERWIRE.md -o THUNDERWIRE.md
```

### Step 2: Activate in your AI

| Tool | How |
|------|-----|
| Claude Code / OpenCode | Copy to project folder, run `opencode` or `claude` |
| Cursor | Settings → Rules for AI → paste full file contents |
| Windsurf | Create `.windsurfrules` → paste full file contents |
| ChatGPT | Settings → Custom Instructions → paste full file contents |
| Gemini / AI Studio | First message of session: paste full file contents |
| Any API | Pass full file as `system` parameter |
| Groq (free) | Use as system prompt with `llama-3.3-70b-versatile` |

### Step 3: Set up your account

```
"My niche is [X]. My audience is [Y]. Run setup."
```

THUNDERWIRE asks 10 questions (including your model, for auto-calibration), then configures itself.

### Step 4: Just talk

```
"Find a viral topic in [niche] and write a thread"
"Write me a tweet about [topic]"
"Review my profile @[handle]"
"Will this perform well? [paste idea]"
"Build me a 7-day content calendar"
"Review this before I post [paste content]"
"Save this" → exports TXT file
```

THUNDERWIRE detects intent and routes to the correct phases automatically.

## The Algorithm Data

X's confirmed 2026 Phoenix transformer scoring weights:

| Engagement | Weight | Notes |
|------------|--------|-------|
| Reply chain (author replies) | +75 | Highest weight — highest leverage |
| Quote tweet | ×25 | Brings new audiences |
| Retweet | ×20 | Pure distribution signal |
| Reply (on your post) | ×13.5 | Conversation = continued push |
| Profile click | ×12 | Drives follows |
| Bookmark | ×10 | Lasting quality signal |
| Like | ×1 | Lowest positive signal |
| Block / Report | -74 | Catastrophic — never risk it |

**Key 2026 algorithm rules:**
- External links near-zero weight — always goes in first self-reply
- 3+ hashtags = -40% reach — hard cap at 2
- Grok sentiment analysis — toxic/negative content suppressed before trending
- SimClusters matching — niche content finds niche audience automatically
- TweepCred threshold of 65 for full distribution
